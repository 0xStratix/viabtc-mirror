# ViaBTC Mirror by 0xStratix

Альтернативные stratum-адреса ViaBTC для майнеров, у которых возникают проблемы с подключением к пулу или теряется часть хешрейта.

## Зеркала подключения

### TCP
- `stratum+tcp://viabtc.0xstratix.com:3333`
- `stratum+tcp://viabtc.0xstratix.com:443`
- `stratum+tcp://viabtc.0xstratix.com:25`

### TLS
- `stratum+tls://viabtc.0xstratix.com:3334`

## Для чего это нужно

Эти зеркала предназначены для пользователей, у которых:

- есть проблемы с доступом к основному адресу пула;
- нестабильное соединение;
- высокий пинг или проблемы с маршрутизацией;
- не доходит часть хешрейта;
- часть шар теряется из-за ограничений сети или провайдера.

Использование альтернативного адреса и другого порта часто помогает стабилизировать соединение и снизить потери хешрейта.

## Как подключить

1. Откройте настройки пула в интерфейсе майнера.
2. Выберите один из адресов выше.
3. Укажите его как основной или резервный stratum-адрес.
4. Проверьте accepted hashrate, stale shares и rejected shares после переключения.

## Какой адрес выбрать

Рекомендуемый порядок проверки:

1. `stratum+tcp://viabtc.0xstratix.com:3333`
2. `stratum+tcp://viabtc.0xstratix.com:443`
3. `stratum+tls://viabtc.0xstratix.com:3334`
4. `stratum+tcp://viabtc.0xstratix.com:25`

Если ваше оборудование поддерживает защищенное подключение, используйте TLS.
